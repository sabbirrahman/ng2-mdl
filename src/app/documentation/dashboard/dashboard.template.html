ng2-mdl

A tiny Angular 2 wrapper for Material Design Lite

Note: This is meant to be a supporting libray until material2 comes out. So I have added alias for selectors as much as close to material2's selectors. Also note that this library will be deprecated once material2 full version is released. So, you are recommended to use the material2 alias selectors. That way migration from ng2-mdl to material2 will be easy ( or should I say less painful? :-/ ).

What is Material Design Lite

Material Design Lite is a small component library for web project that follows google's material design pattern.

Why do I need a wrapper for this?

Material Design Lite is basically developed for vanilla JavaScript based uses. It's not well supported for framework like AngularJS and the next generation version of AngularJS which is know as Angular 2

Ok! But I need a wrapper for Angular 1.x

That's already available here. Angular 2 version of the wrapper is basically base on this angular 1.x version. So, special thanks to Jad Joubran.

How to Use ng2-mdl?

Follow these simple 3 steps:
Step 1: Import

First import the Ng2MdlModule from the library.

import {{'{'}} Ng2MdlModule {{'}'}} from 'ng2-mdl';
Step 2: Inject

Then inject the module in your module's annotation.


@NgModule({{'{'}}
  ...
  imports: [Ng2MdlModule],
  ...
{{'}'}})
Step 3. Use the selector

Now use the selector md-input (materia2 alias) or mdlTextField (camelCase) or mdl-text-field (kebab-case) whichever you like. All the selectors comes in both camel case and kebab case but material2 alias is highly recommanded. (See the table bellow). For example: 


<md-input></md-input> <!-- Recommanded -->
or
<mdlTextField></mdlTextField>
or
<mdl-text-field></mdl-text-field>
